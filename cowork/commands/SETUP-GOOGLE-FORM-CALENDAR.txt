GOOGLE FORM + CALENDAR — embedded ON the website
=================================================

Login: summercrewcyprus@gmail.com

PART 1 — Google Form (embedded in site Step 1)
----------------------------------------------
1. https://forms.google.com → New form
2. Title: Summer Crew Cyprus — Apply

FIELDS (required unless noted):
  • Full name (short answer)
  • Passport number (short answer)
  • Birth year (short answer or dropdown)
  • Parents address (short answer)
  • Mobile / WhatsApp (short answer)
  • Email (short answer)
  • English level (multiple choice)
  • How long in Cyprus? (short answer)

3. Responses → Link to Google Sheet
4. Send → Embed HTML → copy iframe src URL
   Looks like: https://docs.google.com/forms/d/e/XXXXX/viewform?embedded=true
5. Paste into docs/config.js → GOOGLE_FORM_EMBED
6. firebase deploy --only hosting

PART 2 — Google Calendar Appointments (embedded Step 2)
-------------------------------------------------------
1. https://calendar.google.com → Create → Appointment schedule
2. Name: Summer Crew Cyprus — Interview
3. Duration: 20 min
4. Hours: Wed + Thu, 14:00–18:00
5. Copy the public booking page URL
6. Paste into docs/config.js → GOOGLE_CALENDAR_EMBED
7. firebase deploy

Note: Taken slots disappear automatically — others only see open times.
