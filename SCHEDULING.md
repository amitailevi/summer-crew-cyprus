# תיאום ראיונות רצופים — בלי המתנה של שעתיים

## הבעיה
אם נותנים למועמדות "בחרי שעה חופשית" — יוצאות חורים: 10:00, 12:30, 15:00, ואתה מחכה.

## הפתרון: **Calendly** עם slots קבועים + buffer אפס

### שלב 1 — פתיחת חשבון
1. [calendly.com](https://calendly.com) — חינמי מספיק
2. Event type חדש: **"Summer Job Interview — Limassol"**

### שלב 2 — הגדרות האירוע (חשוב!)

| הגדרה | ערך | למה |
|--------|-----|-----|
| **Duration** | **20 דקות** | ראיון סינון — מספיק |
| **Buffer before** | **0** | בלי רווח לפני |
| **Buffer after** | **0** | בלי רווח אחרי |
| **Start time increments** | **20 דקות** | slots: 16:00, 16:20, 16:40… |
| **Max events per day** | 20–25 | שליטה בעומס |
| **Minimum notice** | 2 שעות | לא ב-5 דקות האחרונות |
| **Location** | Leonardo Hotel Limassol (כתובת מלאה) | |

### שלב 3 — שעות זמינות

**שלישי (Tuesday):**
```
16:00 – 20:00
→ 12 slots: 16:00, 16:20, 16:40, 17:00, 17:20, 17:40, 18:00, 18:20, 18:40, 19:00, 19:20, 19:40
```

**רביעי + חמישי (Wednesday + Thursday):**
```
09:00 – 18:00
→ כל 20 דק': 27 slots ליום
```

**שישי:** סגור (או בוקר בלבד — לפי בחירתך)

### שלב 4 — שאלות לפני הראיון (ב-Calendly)

Calendly → Event → **Invitee Questions**:
1. Full name
2. Phone (WhatsApp)
3. Age
4. English level (Basic / Good / Fluent)
5. Available June–September? (Yes / No)

> כך אתה יודע מראש מי מגיעה — בלי שאלון נפרד, אם רוצים פשוט.

### שלב 5 — חיבור לשאלון (Tally / אתר)

**זרימה מומלצת:**

```
מודעת פייסבוק
      ↓
טופס Tally (5 שאלות)
      ↓
דף "תודה" → כפתור: "Pick your interview time"
      ↓
Calendly (רק slots פנויים — אוטומטי!)
```

**איך ב-Tally:**
1. [tally.so](https://tally.so) — טופס חינמי
2. בסוף: **Redirect to URL** = קישור Calendly שלך
3. אופציה: **Hidden fields** — להעביר שם/טלפון ל-Calendly (אם משלמים / Cal.com)

**חלופה פשוטה יותר:** מודעה → ישר ל-Calendly (בלי Tally). השאלות רק ב-Calendly.

### שלב 6 — למנוע no-shows

- Calendly שולח **SMS/Email reminder** אוטומטי (24h + 1h)
- הוסף ב-Description: "Please arrive 5 min early. WhatsApp if you're late: [מספר]"
- אחרי booking — **WhatsApp ידני** ל-3 מועמדות הראשונות: "See you Tuesday 16:20 ✨"

### שלב 7 — יום הראיונות

```
16:00 – מועמדת 1
16:20 – מועמדת 2   ← רצוף, בלי המתנה
16:40 – מועמדת 3
...
```

אם slot נשאר ריק — **5 דקות break** לך. עדיף מ-2 שעות ריקות.
